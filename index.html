Unity与Android交互—aar

AndroidStudio版本：4.1.1

Unity版本：2017.3.1.p4

AndroidStudio导出aar文件

1.新建项目

Create New Project – Empty Activity – Next

  • Name的名称随意无关紧要

  • Minimus SDK是用来选择支持的最低版本（建议最低选择API 16）

– Finish

2.新建库

在app目录上右键 – New – Module – Android Library

  • Module name就是库名

  • Bytecode Level切记不可选8，选7和6都可以（否则在Unity中Build时会报错）

– Finish

3.导入Unity中的classes.jar

  • classes.jar在对应版本的Unity目录下的Editor\Data\PlaybackEngines\AndroidPlayer\Variations\mono\Release\Classes

  • 将其导入Module的libs文件中（切换到Project视图下可见）

  • 右键jar包 – Add As Library – OK

4.新建类

库文件 – java –（第一个文右键）– New – Java Class

  • 取一个类名后面在Unity调用会用到

5.为类添加方法

在类下添加方法，添加的方法可以在Unity中调用，如：Add方法

6.生成aar文件

  1. 选中Module文件点击Build – Make Moudle

  2. 在Mandle文件夹下build – outputs – aar找到aar文件将其拖到桌面（Project视图）

  3. 使用解压缩软件打开删除libs下的classes.jar文件

1.将aar文件放入Unity

打开Unity在资源目录下新建 Plugins/Android 把aar文件放入Android目录下

安卓的三种BuildSystem方式

前言

  1. 下载AndroidBuildStudio（在Unity中添加模块）（在Unity2019以上版本可以顺便添加Android SDK & NDK以及JDK，如果添加则可以跳过前言）
  2. 下载JDK-1.8版本 配置环境变量 需要javac -version显示版本正确，而不是java -version
  3. 下载SDK：可以在下载AndroidStudio时下载
  4. 下载NDK： 可以在Unity中点击Edit–Preferences–External Tools中在NDK右侧下载官方版
  5. 创建密钥：打开命令提示符进入安装的java–bin目录在输入 然后按照要求输入创建即可，可将文件放入想要放置的位置，切记记住密码。

一、UnityRemote

  1. 安卓机安装UnityRemote软件
  2. 下载AndroidStudio中的Google USB Driver（下一种方法有下载方式）
  3. 在Unity中Edit – Project Settings – Editor – Device更改设置为Any Andorid Device
  4. 手机打开UnityRemote，在Unity中直接运行即可

二、AndroidStudio-AVD

  1. 电脑上安装AndroidStudio
  2. 安装时勾选安装AVD选项（默认勾选）
  3. 下载安装后双击打开软件一般情况会下载gradle，但由于网络会下载失败，这时复制下载网址

    下载压缩包，将压缩包（不解压）放入C:/Users/(用户名目录)/.gradle/wrapper/dists/(压缩包对应的版本号目录)/(乱码目录)文件夹下，返回AndroidStudio等待软件自动解压完成
  4. 点击File–Settings–Appearance&Behavior-SystemSettings–AndroidSDK下载SDK
    以及镜像文件
  5. 下载SDKTools–Google USB Driver
  6. 全部下载完成之后点击Tools–AVD Manager–Create Virtual Device后选择
    –next
    –next–Finish
  7. 点击三角启动手机

三、模拟器

  1. 如夜神模拟器
  2. 直接将Apk文件拖入模拟器中运行即可

博客搭建Hexo + GitHub

安装node.js

官方网址：http://nodejs.cn/

cnpm安装-淘宝镜像下载

npm install -g cnpm  --registry=https://registry.npm.taobao.org

hexo下载

cnpm install -g hexo-cli

新建文件夹

mkdir Blog

初始化hexo

安装git：https://git-scm.com/download/win
管理员用户进入Blog执行

hexo init //新建一个网站

安装git部署插件

cnpm install --save hexo-deployer-git

在git中添加新存储库

库名使用 <用户名>.github.io

修改配置

打开_config.yml最下面修改并添加

    type: git
    repo: <github中添加的库的网址>
    branch: master

打开hexo

hexo n "文件名" //新建一篇文章
hexo g //生成静态文件
hexo s //启动服务器

部署hexo

hexo d //部署服务器

官方参考地址

https://hexo.io/

Unity中安卓的三种BuildSystem方式

2021-01-08
Unity
  1. 前言
  2. 一、UnityRemote
  3. 二、AndroidStudio-AVD
  4. 三、模拟器

前言

  1. 下载AndroidBuildStudio（在Unity中添加模块）（在Unity2019以上版本可以顺便添加Android SDK & NDK以及JDK，如果添加则可以跳过前言）
  2. 下载JDK-1.8版本 配置环境变量 需要javac -version显示版本正确，而不是java -version
  3. 下载SDK：可以在下载AndroidStudio时下载
  4. 下载NDK： 可以在Unity中点击Edit–Preferences–External Tools中在NDK右侧下载官方版
  5. 创建密钥：打开命令提示符进入安装的java–bin目录在输入 然后按照要求输入创建即可，可将文件放入想要放置的位置，切记记住密码。

一、UnityRemote

  1. 安卓机安装UnityRemote软件
  2. 下载AndroidStudio中的Google USB Driver（下一种方法有下载方式）
  3. 在Unity中Edit – Project Settings – Editor – Device更改设置为Any Andorid Device
  4. 手机打开UnityRemote，在Unity中直接运行即可

二、AndroidStudio-AVD

  1. 电脑上安装AndroidStudio
  2. 安装时勾选安装AVD选项（默认勾选）
  3. 下载安装后双击打开软件一般情况会下载gradle，但由于网络会下载失败，这时复制下载网址

    下载压缩包，将压缩包（不解压）放入C:/Users/(用户名目录)/.gradle/wrapper/dists/(压缩包对应的版本号目录)/(乱码目录)文件夹下，返回AndroidStudio等待软件自动解压完成
  4. 点击File–Settings–Appearance&Behavior-SystemSettings–AndroidSDK下载SDK
    以及镜像文件
  5. 下载SDKTools–Google USB Driver
  6. 全部下载完成之后点击Tools–AVD Manager–Create Virtual Device后选择
    –next
    –next–Finish
  7. 点击三角启动手机

三、模拟器

  1. 如夜神模拟器
  2. 直接将Apk文件拖入模拟器中运行即可

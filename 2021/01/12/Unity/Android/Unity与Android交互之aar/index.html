Android导出aar文件

1. 创建AndroidStudio项目

  • 项目的名称随便写，后面用不到
  • Minimus SDK选择最低即可

2. 新建Library

在Project窗口右键 – New – Module – Android Library

  • Module name为库名
  • Bytecode Level使用 7 或者 6 （否则在最后Unity中Build时会报错）

3. 添加Unity的classes.jar包到libs文件夹下（Project试图可见）并右键 – Add as library添加到当前库

  • classes.jar在Unity的安装目录下的\Editor\Data\PlaybackEngines\AndroidPlayer\Variations\mono\Release\Classes

4. 新建 Activity 或者 Class

  • Activity 使用略微复杂，但可以使用Android提供的一些功能，如Toast

  • Class 使用简单，但只能写一些自己的方法（也可以写一些方法来实现Toast提供一篇大佬的文章- 地址连接）

  • 新建Activity或Class选一种即可
